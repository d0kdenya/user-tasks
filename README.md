## Перед запуском

Перед запуском важно заполнить конфиг (.development.env)
P.S. можно использовать поля из example.env

## Загрузка зависимостей

```bash
$ npm install
```

## Запуск приложения

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Важные моменты
API запросов описана в Swagger API, которая доступна по адресу:
${ URL }:${ PORT }/api
